Event-driven transaction pipeline via **RabbitMQ**, with **Resilience4j** circuit breakers for fault-tolerant, idempotent ledger updates.

`Java` `Spring Boot` `Next.js` `RabbitMQ` `Redis` `MySQL` `Razorpay API`
